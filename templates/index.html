О Python

1. Основная информация

Python - многофункциональный язык высокого уровня довольно простого синтаксиса, понятного даже новичку в программировании.

2. Синтаксис

Основные классы данных

Как такового типа данных в Python'е нет, все объекты разделяются на классы, т.е. класс "int", класс "bool", класс "str". Объекты подобных классов очень похожи на переменные одноименных типов данных в C++. Примеры использования:
something = 5 - теперь переменная something является объектом класса Int
something = '123' - теперь переменная something является объектом класса Str

Динамическая типизация

В Python'е использует динамическая типизация данных, а это значит, что тип данной переменной МОЖЕТ ИЗМЕНЯТЬСЯ, как это и было показано в примере выше!
Основные типы(классы) данных в python'е:
str #'123'
float #1.4
int #5
bool #True
list #[1, 2, 3] - объекты класса поддерживают итерацию, подобны массиву данных в C++, однако не имеют ограничений по размеру по размеру!
tuple #(1, 2, 3) - аналогично list, но запрещенно изменять содержимое его объектов каким-либо способом!

Конструкции

Основные конструкции языка

if condition: ...
elif contition: ...
else: ...
- условный оператор ветвления, представлен также, как и в других языках, например, в Си

for i in something:
...
- цикл, выполняющийся, пока не дойдет до края something(где something - итератор). Доступ к текущему рассматриваемому значению получается с помощью переменной цикла i(можно и любое другое название)

while condition:
... - цикл, выполняющийся, пока выражение condition истинно

Функции

Основные функции языка

print(something, ..., end='\n', sep=';', file=sys.stdout) - выводит переменные/значения/..., которые описываются в бесконечных параметрах end - чем окончить вывод, sep - чем разделить выражения для вывода, file - в какой файл нужно вывести, по умолчанию - в стандартный поток вывода

input(message) - возвращает строку, полученную из стандартного потока ввода, необязательный параметр message - что нужно вывести перед вводом, например, подсказку

map(func, iterable, ...) -> iterator - возвращает итератор, полученный путем применения функции func ко всем элементам последовательностей, описанных во втором, бесконечном параметре

filter(func, iterable, ...) -> iterator - возвращает итератор, полученный путем применения функции func ко всем элементам последовательностей, описанных во втором, бесконечном параметре. Элементы последовательностей, при которых func вернула True, будут записаны в итератор

dict([object]) -> dictionary - возвращает словарь, полученный путем преобразования object в словарь

set([object]) - возвращает множество, полученное путем преобразования object в множество

str([object]) - возвращает строковое представление оbject

Как представленно объектно-ориентированное программирование в Python'е

Объявление класса

Класс объявляется следующей констркуцией:
class name(...):
def __init__():...
- где name - имя класса, бесконечные параметры в скобках - наследуемые классы, т.е. классы, методы которых необходимо получить(подробнее о наследовании здесь) Конструктор класса __init__(self, ...) обязателен, если вы не наследовали какие-либо классы, он отвечает за первоначальные объявления, которые будут выполнены в ходе объявления объектов класса.
Пару слов о self: self - обязательный параметр любых методов класса в python'е. Он - объект класса, который мы рассматриваем. Таким образом в одних методах класса мы можем вызывать другие, обращаться к каким-либо переменным объекта. Например, print(self.param) - выведет на экран текущее значение параметра param.

Методы класса

В Python'е метод можно определить и/или перегрузить таким образом: def name(self, param1, *args, **kwargs):
...
 - Где name - имя метода, self - обязательный параметр, далее идут параметры, как и при обычных функциях. Чтобы получить доступ к методам наследуемого класса, нужно использовать self.__super__()
